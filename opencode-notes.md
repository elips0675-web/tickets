## Правила работы
1. **Никогда не удаляй существующие импорты** — только добавляй новые.
2. После любого изменения файла проверь `npx tsc --noEmit`.
3. Если пользователь жалуется на белый экран — сначала прочитай ошибку в консоли браузера, потом чини.
4. Не редактируй импорты вручную — если нужно добавить символ, дополни существующую строку, а не переписывай её.
5. Перед edit всегда читай актуальное содержимое файла (уже сделано).
